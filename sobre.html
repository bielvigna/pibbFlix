Sobre


Esse site foi desenvolvido como parte de um trabalho de HTML e CSS, com o objetivo de criar uma plataforma de streaming fictícia chamada "VignaFlix". O site apresenta uma interface simples e intuitiva, com seções para filmes mais vistos e recomendados. Cada filme possui uma imagem, título e link para mais informações. Foi um trabalho árduo, realizado com dedicação para aplicar os conhecimentos adquiridos.
